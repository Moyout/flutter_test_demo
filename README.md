# flutter_demo

A Flutter test project. Example of filtering and sorting

## Start Run Project
 *Prerequisite: Flutter SDK*
```bash
sdks:
  dart: ">=3.5.4 <4.0.0"
  flutter: ">=3.24.0"
```
Clone Project
```bash
  git clone https://github.com/Moyout/flutter_test_demo.git
```
Get dependency packages
```bash
  cd flutter_test_demo
  flutter pub get 
```
Run
```bash
  flutter run
```
